# COVID-19 Data Analysis using Python

## Project Overview

This project analyzes COVID-19 data using Python. The dataset was cleaned, processed, and visualized to identify trends in confirmed cases, deaths, testing, and vaccinations.

## Objectives

* Perform data cleaning and preprocessing
* Handle missing values
* Analyze COVID-19 trends over time
* Visualize cases, deaths, testing, and vaccination progress
* Extract meaningful insights from the dataset

## Tools & Libraries Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Data Cleaning Steps

* Checked dataset structure and summary statistics
* Converted date column to datetime format
* Handled missing values
* Verified data consistency

## Exploratory Data Analysis

The analysis includes:

* Daily confirmed cases trend
* Daily deaths trend
* Testing statistics
* Vaccination progress over time
* Correlation analysis using heatmaps

## Key Insights

* COVID-19 cases showed multiple waves over the observed period.
* Vaccination coverage increased significantly over time.
* Strong relationships were observed between several COVID-related metrics.
* Visualizations helped identify major trends and patterns in the data.
